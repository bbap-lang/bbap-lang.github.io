abap-dependencies

The abap-dependencies value is an array of paths to files that include the default ABAP/SAP values, like tables or types.
For example if you want to use the table 'VBAK' you have to add the path to the file that includes the table. That way, you can easily use the table in your code.

Example

{ "abap-dependencies": [ "abap/vbak.abap" ] }
