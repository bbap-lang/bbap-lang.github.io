use-scopes

The use-scopes value in the config file describes if the compiler should add additional scopes or just use the default abap ones.

Examples

True

let i = 0; if(i == 0){ let i = 1; println(i); // 1 } println(i); // 0

The use-scopes value is true by default. This means that the compiler will add a new scope for every block.

False

let i = 0; if(i == 0){ let i = 1; // Error: let is here not allowed, because 'i' is already defined in the parent scope println(i); // 1 } println(i); // 1

Allowed Values

Value Description
True Additional scopes are added.
False No additional scopes are added.
