BBAP - Home

BBAP is in a very early stage of development. So there are many features missing and the language is not stable. If you want to contribute to the project, you can find the source code at GitHub. Any help is appreciated!

This is the official documentation for the BBAP programming language. BBAP is designed for the SAP environment and compiles to ABAP. This means that you can use BBAP to write ABAP code.
BBAP is designed to be more modern and easier to use than ABAP.

If you don't work with SAP, you currently can't use BBAP!

If you want to start using BBAP, you can find more information in the Getting Started section.
