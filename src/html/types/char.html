Char

The char type is used to store either a single character or a string of characters. The difference between the char and string type is that the char has a fixed size. In a table only chars can be stored, not a dynamic string. The size of a char is fixed to the variable, meaning that you can't change the size of a char after it has been declared.

A char is always implicit castable to a string, but a string is not implicit castable to a char. For that you have to use the ToCharArray function.
More information about string can be found here.

Example

let str: char[14] = 'Hello World';

In this example the variable str is declared as a char with the size of 14. The size of the char is set to 14.
The string literal 'Hello World' is 11 characters long. The compiler will automatically fill the remaining 3 characters with spaces.
