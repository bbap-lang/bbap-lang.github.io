Tables

The tables keyword is used to declare a table. A table is a collection of values.
The values of a table can be changed. The values of a table can be accessed with SQL.

let exampleTable: Table<int>; exampleTable.append(1); exampleTable.append(2); exampleTable.append(3); let ones: Table<int>; ones = SELECT * FROM exampleTable WHERE exampleTable = 1;

The Table keyword is used to declare a table. The Table keyword takes one type as a parameter.
The type of the table is the type of the values in the table.
The append method is used to add a value to the table.

Examples

Append

The append method is used to add a value to the table, it takes one parameter.

let exampleTable: Table<int>; exampleTable.append(1); exampleTable.append(2); exampleTable.append(3);
