Field Symbols

A field symbol is used to access one value in a table. It references the value, so if the value changes, the field symbol will change too, and vice versa.

Example

let exampleTable: Table<int>; let exampleFieldSymbol = exampleTable.ToFieldSymbol();

The ToFieldSymbol method is used to create a field symbol from a table.
The ToFieldSymbol method takes no parameters.

Additionally, field symbols are used in foreach loops.
