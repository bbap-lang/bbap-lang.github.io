Foreach

The foreach loop is used to iterate over a table.

let exampleTable: Table<int>; foreach (let exampleFieldSymbol on exampleTable) { // Do something }

The variable exampleFieldSymbol is a field symbol that references a value in the table exampleTable.

With every iteration, the field symbol will reference the next value in the table.

The let keyword is used to declare a variable. The let is optional, you can also use a field symbol that is already declared.

let exampleTable: Table<int>; let exampleFieldSymbol: FieldSymbol<int>; foreach (exampleFieldSymbol on exampleTable) { // Do something }
