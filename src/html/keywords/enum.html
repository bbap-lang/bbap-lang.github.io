Enum

An enum is a type that contains a set of named values. For example, the days of the week can be represented by an enum.

Enums are declared with the enum keyword. After the keyword you have to specify the name of the enum. The values of the enum are defined in a block. The values are seperated by a comma.

Examples

enum Days { Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday }

Usage

An enum value is used like a static variable.

Example

let day = Days::Monday;

Comparing

Enums can be used like the underlying type. That means you can compare them with the == operator, but you can also check if the value is smaller or greater.

Example

let day = Days::Monday; if (day == Days::Monday) { printLine("It's Monday!"); }

Custom Values

By default, the values are of the type int, they start at 0 and increment by 1 for each value.

You can define your own values. They are specified with a = after the name of the value.

Example

enum Days { Monday = 1, Tuesday = 2, Wednesday = 3, Thursday = 4, Friday = 5, Saturday = 6, Sunday = 7 }

Using a different type

Normally, the values of an enum are of the type int. You can change the type of the values by specifying it after the name of the enum.

>Example

> enum Days: string { Monday = "Mon", Tuesday = "Tues", Wednesday = "Wed", Thursday = "Thurs", Friday = "Fri", Saturday = "Sat", Sunday = "Sun" }
