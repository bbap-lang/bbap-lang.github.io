Func

Functions are used to group code together. They can be called from anywhere in the code.

Declaration

Functions are declared with the func keyword. After the keyword you have to specify the name of the function. After that you have to specify the parameters of the function in a block.
The parameters are defined like variables, but without the let keyword and seperated by a comma.

Functions can have one or multiple return values. The return values are defined after the parameters.
If the function only has one return value, it can stand by it self. Multiple must be grouped in a block of parentheses and seperated by comma.

Example

func add(a: int, b: int): int { return a + b; } func subtract(a: int, b: int): (int, int) { let firstResult = a - b; let secondResult = b - a; return (firstResult, secondResult); }

Calling

When you call the funtion you have to specify the name of the function. After that you have to specify the arguments of the function.
If you have multiple return types, you have again to wrap them in a block of parentheses.

Example

let result = add(1, 2); // 3 let (firstResult, secondResult) = subtract(1, 2); // -1, 1
