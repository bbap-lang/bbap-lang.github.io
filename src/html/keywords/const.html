Const

The const keyword is used to declare a constant. A constant is a value that cannot be changed.
The value of a constant must be set when it is declared. But it can be obtained in any way, like from a function.

The value of a constant cannot be changed. If you try to change the value of a constant, an error will be thrown.

const value = 1; printline(value); // 1 value = 2; // Error
