Switch/Case

The switch keyword is used to compare a value to multiple values. It can replace multiple if-statements.

switch (value) { case 1 { printline("One"); } case 2{ printline("Two"); } default { printline("Unknown"); } }

In this example, the value variable is compared to the values 1 and 2. If the value is 1, the One string will be printed. If the value is 2, the Two string will be printed. If the value is neither 1 nor 2, the Unknown string will be printed.

The default keyword is optional. If the value is not equal to any of the values in the case statements, the code in the default statement will be executed.

The values in the case statements can be any value, not just numbers. For example, you can compare a string to multiple strings.

switch(value){ case "One" { printline(1); } case "Two" { printline(2); } }
