Println

The println function is used to print a value to the console. The function takes either one argument or no argument. If the function takes one argument, the value gets printed. The value can be of any type.

Additional to the normal print function, the println function prints a new line after the value is printed. If the function takes no argument, only a new line gets printed.

Example

println("Hello World!"); // Hello World! println(); // new line
